LEVEL 2 – Basic Algorithms & Data Structures

ID:
Level ini saya gunakan untuk melatih cara berpikir langkah demi langkah dan memperkuat pemahaman struktur data dasar. Saya berlatih menggunakan list, dictionary (hash map), set, serta stack/queue sederhana, sambil mengimplementasikan algoritma seperti pencarian linear, mencari nilai minimum/maksimum, menghitung frekuensi, dan sorting sederhana (misalnya bubble atau selection sort). Tujuan saya di level ini adalah membangun intuisi yang kuat dalam memilih struktur data dan menulis solusi yang logis dan mudah dibaca.

EN:
In this level I focus on training step‑by‑step problem‑solving and deepening my understanding of basic data structures. I work with lists, dictionaries (hash maps), sets, and simple stacks/queues while implementing algorithms such as linear search, finding minimum and maximum values, counting frequencies, and simple sorting (for example bubble or selection sort). The goal of this level is to build strong intuition for choosing data structures and to write solutions that are logical and easy to read.
