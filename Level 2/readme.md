Saya berlatih mengolah list, dictionary, dan set lewat tugas seperti mencari nilai maksimum/minimum/rata‑rata secara manual, mengelola stok barang dengan dict, dan menunjukkan bagaimana set menghilangkan duplikasi data. Saya juga mengimplementasikan sendiri konsep stack dan queue menggunakan list atau deque serta mensimulasikan kasus nyata seperti tumpukan piring dan antrian loket. Latihan ini saya lanjutkan dengan soal‑soal kecil seputar filter list, manipulasi string, dan penggunaan dict sebagai hash map untuk menghitung frekuensi huruf dan kata. Sebagai penutup, saya membangun tiga mini‑project: analisis teks sederhana, sistem penilaian kelas, dan simulasi antrian layanan berbasis queue, lalu mencoba merapikan salah satunya menjadi lebih modular dengan beberapa fungsi terpisah. Tujuan saya di level ini adalah memiliki intuisi yang kuat ketika memilih dan memakai struktur data, sekaligus merasa nyaman menyusun algoritma sederhana dari awal sampai selesai.

EN:
In this second level I focus on training my algorithmic thinking and deepening my understanding of basic data structures. I practice working with lists, dictionaries, and sets through tasks such as manually finding maximum, minimum, and average values, managing an inventory using a dictionary, and demonstrating how sets remove duplicate elements. I also implement stack and queue behavior using lists or deque and simulate practical scenarios like a stack of plates and a ticket counter queue. On the algorithm side, I write my own linear search function, do simple counting exercises (counting occurrences of numbers and vowels), and implement bubble sort and selection sort to understand how sorting works step by step. I extend this with small problems involving list filtering, string processing, and using dictionaries as hash maps to count character and word frequencies. Finally, I build three mini projects: a simple text analyzer, a classroom grading system, and a service‑queue simulator, then refactor at least one of them into a more modular, function‑based design. The goal of this level is to develop strong intuition for choosing and using data structures while feeling confident in assembling simple algorithms from start to finish.
