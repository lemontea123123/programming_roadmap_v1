Di Level 3 ini saya belajar dasar pengolahan data dengan Python (Pandas + sedikit NumPy) dan SQL. Fokusnya: baca data dari CSV, bereskan data (cek dan tangani nilai kosong), buat kolom baru seperti total_penjualan, rangkum dengan statistik sederhana dan groupby (per produk/per bulan), lalu buat grafik garis dan batang sederhana. Saya juga berlatih SQL dasar di SQLite: membuat tabel penjualan, menulis query SELECT + WHERE, ORDER BY, LIMIT, fungsi agregasi, GROUP BY, dan JOIN dengan tabel produk, lalu menerapkannya dalam mini project laporan penjualan dan “query explorer” kecil. Selain itu saya mulai menyentuh NumPy dasar untuk memahami array, operasi vektor sederhana, dan bagaimana NumPy dipakai bersama Pandas untuk transformasi numerik di kolom DataFrame.

EN:
In Level 3 I learn the basics of working with data using Python (Pandas + a bit of NumPy) and SQL. The focus is reading CSV files, cleaning data (checking and handling missing values), creating new columns like total_sales, summarizing with simple statistics and groupby (per product/per month), and making basic line and bar charts. I also practice core SQL in SQLite: creating a sales table, writing SELECT + WHERE, ORDER BY, LIMIT, aggregate functions, GROUP BY, and JOIN with a products table, then using all of this in a small sales report mini project and a simple query explorer. In addition, I start using basic NumPy to understand arrays, simple vectorized operations, and how NumPy works together with Pandas for numeric transformations on DataFrame columns.
