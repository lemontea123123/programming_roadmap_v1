LEVEL 5 – Integrasi API AI

ID:
Di Level 5 ini saya belajar memakai API AI (LLM) langsung dari Python dan menghubungkannya ke aplikasi catatan sederhana. Saya berlatih konsep HTTP dan API (GET/POST, JSON, API key), lalu menggunakan library requests untuk mengirim prompt ke endpoint AI dan membaca respons teks. Dari situ saya membuat beberapa script kecil untuk merangkum teks dan memperbaiki grammar bahasa Inggris, kemudian menambah fitur “Ringkas” dan “Cek grammar” ke app catatan berbasis CLI. Saya juga melatih alur baca file teks → kirim ke API → simpan ringkasan ke file lain, termasuk versi batch untuk banyak file sekaligus. Tujuan saya di level ini adalah paham alur end-to-end: dari Python mengirim request ke layanan AI, menerima hasilnya, dan memakainya sebagai bagian dari workflow aplikasi yang nyata.

EN:
In Level 5 I learn how to use AI (LLM) APIs directly from Python and connect them to a simple notes application. I practice core HTTP and API concepts (GET/POST, JSON, API keys) and use the requests library to send prompts to an AI endpoint and read text responses. From there I build small scripts for text summarization and English grammar correction, then add “Summarize” and “Check grammar” features to a CLI-based notes app. I also practice the flow of reading a text file, sending its contents to the API, and saving the summary to another file, including a batch version that processes multiple files in a folder. The goal of this level is to understand the full end-to-end flow: Python sending requests to an AI service, receiving the results, and integrating them into a real application workflow.
